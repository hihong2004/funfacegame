# Face Crush

> **Match-3 消除遊戲 - 自訂照片與主題的臉部粉碎遊戲**

## 🎮 遊戲簡介

Face Crush 是一款創新的 Match-3 消除遊戲，玩家可以上傳自己或朋友的照片，結合不同主題（生日、浪漫、派對等），在遊戲中消除配對的臉孔，並享受客製化的過關特效。

## ✨ 特色功能

- ✅ **自訂照片**：上傳最多 6 張照片或使用預設表情符號
- ✅ **多種主題**：生日、浪漫、派對、寵物、惡搞
- ✅ **Match-3 消除**：經典三消玩法
- ✅ **客製化特效**：過關時顯示自訂照片與主題特效
- ✅ **關卡系統**：10 個漸進式關卡
- ✅ **分享功能**：分享到 Instagram、TikTok、Facebook
- ✅ **進度追蹤**：即時分數與進度條
- ✅ **響應式設計**：支援桌面與行動裝置

## 🎯 遊戲玩法

### 基本規則

1. **拖曳交換**：拖曳相鄰的方塊進行交換
2. **三個配對**：連成三個或以上相同顏色/照片的方塊會消除
3. **累積分數**：每消除一個方塊得 1 分
4. **達成目標**：達到目標分數即可過關

## 🔧 技術規格

### 依賴模組

- **核心模組**
  - `core/storage.js` - 資料儲存
  - `core/analytics.js` - 數據追蹤
  - `core/config.js` - 全域配置

### 遊戲設定

```json
{
  "boardSize": 8,              // 8x8 棋盤
  "maxPhotos": 6,              // 最多 6 張照片
  "slotColors": 6,             // 6 種顏色
  "gameSpeed": 100,            // 遊戲速度（毫秒）
  "vibrationEnabled": true     // 震動回饋
}
```

## 📊 數據追蹤

遊戲會追蹤以下事件：

- `game_start` - 遊戲開始
- `level_complete` - 關卡完成
- `share` - 分享事件
- `match` - 消除事件（包含配對數量）

## 🚀 部署

### 本地開發

1. 確保所有核心模組已載入
2. 在 `config.js` 中啟用此遊戲
3. 開啟瀏覽器訪問 `/plugins/face-crush/index.html`

### 生產環境

1. 合併到 `main` 分支
2. Vercel 自動部署
3. 遊戲將顯示在大廳
